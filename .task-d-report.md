## Concerns

- **Node 25 teardown crash**: `FATAL ERROR: v8::ToLocalChecked Empty MaybeLocal`
  fires after vitest exits. This is a pre-existing bug in Node 25 / v8's
  `cjsPreparseModuleExports` during worker teardown (exit code 134). It was
  present before this change (confirmed by stashing and rerunning). All tests
  themselves pass and complete before the crash.

- **dist/ tracked in git despite .gitignore**: The repo commits pre-built dist
  artifacts. `npm install` removed them from disk; restored via `git checkout -- dist/`
  before committing so the snapshot is unchanged. The dist does not include the
  new `verify.ts` output — a `npm run build` is needed before publishing.

- **No signature verification**: By design (matches Rust SDK split). Callers
  must verify against the JWKS (`client.orgJwks(orgUuid)`) before trusting claims.
